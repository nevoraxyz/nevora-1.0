# Nevora v1.0 - 2025-12-04

**Description:**  
Nevora 1.0 introduces the world's first **ZK Health Identity** — a wallet-based personal health identity that allows users to verify their health journey without disclosing personal data. This version lays the foundation for a **secure, private, and transparent** user experience.

**Key Features:**  
- Private, wallet-based health profile  
- Verify health journey without revealing personal information  
- Interactive dashboard with score & coins display  
- Modern UI with gradient backgrounds and blob animations  
- Foundation for future exchange & reward features  

**Improvements & Changes:**  
- Initial project structure ready for updates  
- Responsive layout for desktop & mobile  
- Frontend performance optimization and reusable components  

**Notes:**  
This is the first release (v1.0) — focused on **privacy, security, and usability**. Upcoming versions will expand exchange, reward, and additional health tracking features.
